# 🌤️ WeatherScope Pro

A modern, responsive weather dashboard built with vanilla JavaScript, featuring real-time data and immersive visual effects.

## ✨ Features
- **Real-time Data:** Fetches live weather information using OpenWeatherMap API.
- **Glassmorphism UI:** Modern frosted-glass design with CSS backdrop filters.
- **Dynamic Atmosphere:** 
  - Background images change based on the weather (Clear, Clouds, Rain, Snow).
  - Immersive particle effects (falling rain/snow) using **tsParticles**.
- **Interactive Elements:** 
  - 3D Tilt effect on the weather card using **Vanilla-Tilt.js**.
  - Animated weather icons (spinning sun, drifting clouds).
- **Dark/Light Mode:** Toggle between themes with smooth transitions.
- **Motivational Quotes:** Dynamic quotes tailored to current weather conditions.

## 🛠️ Built With
- HTML5 & CSS3 (Custom Variables & Keyframes)
- JavaScript (ES6+, Async/Await, Fetch API)
- [OpenWeatherMap API](https://openweathermap.org/)
- [tsParticles](https://particles.js.org/)
- [FontAwesome](https://fontawesome.com/)
